# Code of Conduct

This project follows a simple rule set:

- Be respectful and direct
- Assume good intent, but prioritize clarity
- Critique code and ideas, not people
- Do not post secrets, access tokens, Claude session links, or private logs in public threads

Maintainers may remove comments, issues, PRs, or contributors that make collaboration unsafe or unproductive.

For security-sensitive or conduct-sensitive issues, use the guidance in [SECURITY.md](./SECURITY.md).
